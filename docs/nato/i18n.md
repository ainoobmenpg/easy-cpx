# UI / i18n 指針

この文書はOperational CPXのUI文言・多言語対応方针を定義する。

---

## 1. 基本方針

### 1.1 表記言語

- **基本**: 日本語
- **タイトル・見出し**: 日本語
- **ボタン・ラベル**: 日本語
- **NATO略語**: 英語大文字（SITREP, INTSUM, OPSUM等）
- **用語**: [glossary.md](./glossary.md) に従う

### 1.2 UI基本原则

| 原則 | 内容 |
|------|------|
| 視認性 | 画面を見れば状況が分かる |
| 名称一貫性 | 同じ意味の言葉は同じ |
語統一表記| 略 | NATO標準略語を使用 |

---

## 2. i18n構造

### 2.1 ファイル配置

```
frontend/
  public/
    locales/
      ja/
        translation.json   # 日本語（主）
      en/
        translation.json   # 英語（拡張用）
  app/
    lib/
      i18n.ts            # i18nフック
```

### 2.2 translation.json構造

```json
{
  "common": {
    "retry": "再試行",
    "gameTitle": "作戦級CPX"
  },
  "tabs": {
    "info": "情報",
    "history": "履歴"
  },
  "commands": { ... },
  "unitTypes": { ... }
}
```

### 2.3 アクセス方法

```typescript
import { useTranslation } from '../lib/i18n';

function MyComponent() {
  const { t } = useTranslation('ja');
  return <button>{t('common.retry')}</button>;
}
```
