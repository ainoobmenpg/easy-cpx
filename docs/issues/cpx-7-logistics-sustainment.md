# CPX-7: 兵站モデル（Class III/V、補給路、再補給サイクル）

### 背景
ammo/fuelの3段階はあるが、MOT/路線/拠点/輸送の概念がなく、作戦継続性の表現が不十分。

### 提案
- データ: `supply_nodes`/`supply_routes`/`convoys`、在庫（Class III/V）、MOT/補給所、再補給ターン。
- 連動: 補給路寸断や在庫不足を前提チェックへ反映。LOGSITREPにステータス集計を表示。
- UI: `Sustain`タブで補給可視化（在庫、ルート状況、次回補給予定）。

### 受け入れ基準
- ルート寸断で弾薬/燃料供給が遅延/停止し、判定や行動に影響する。
- LOGSITREPに在庫/輸送ステータスが反映される。
