# CPX-4: MEL/MIL（Inject）システム＋EXCONパネル

### 背景
イベントデッキはあるが訓練用のMEL/MIL（Master Events/Inject List）が未整備。白セルが意図通りに注入・観察・評価できない。

### 提案
- データ: `injects` テーブル（目的、発動条件、観察項目、評価ポイント、影響範囲、想定対応）。
- 運用: EXCON UIから条件注入/即時注入。注入ログと効果ログをTurnに紐付け。
- 連動: 影響は審判前提（通信断/EW/補給寸断等）の修正値として適用。

### 受け入れ基準
- MEL/MILを登録/実行/追跡でき、注入の影響が判定/レポートに反映される。
- 監査ログで全Injectの時刻・効果が追える。
