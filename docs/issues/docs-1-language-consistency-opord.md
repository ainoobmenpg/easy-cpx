# Docs-1: OPORD既定文の言語混在（簡体字/誤用）修正

### 問題
`backend/app/services/opord_service.py` の `create_default_opord()` で生成される既定文に、簡体字や日本語として不自然な語が混在しています。
- 例:
  - 「直射火力中队」→「直射火力中隊」
  - 「敵渗透部隊」→「敵浸透部隊」
  - 「的一般補給は後方より」→「一般補給は後方より」
  - 「衛生班（1個分身）」→「衛生班（1個分隊）」または「衛生小隊」
  - 「輸送中队」→「輸送中隊」
  - 「後送は戦場病院へ」「後方医院へ空送」→「後送は戦場病院へ」「後方病院へ空輸」
  - 「後方支援大队」→「後方支援大隊」

### 影響
- 公式ドキュメント/ゲーム内文面の信頼性を損なう。
- NATO CPXを志向する上で用語整合性が低下。

### 対応
- 上記箇所を純日本語の軍事用語に修正。
- 用語集（Docs-6）に準拠し、ユニット編制語（班/分隊/小隊/中隊/大隊/旅団）を統一。
- PR時にlint的な簡体字検出（正規表現）を走らせるCIを提案。

### 完了条件
- 上記語句がすべて修正され、ユニット編制の粒度表記が統一される。
- 追加で全リポジトリをスキャンし、簡体字/中英混在が0件であることを確認。
