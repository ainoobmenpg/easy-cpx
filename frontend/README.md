# Operational CPX - Frontend

Next.js frontend for the Operational CPX military simulation game.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

```
app/
├── game/           # Main game UI
├── scenarios/      # Scenario selection
├── debriefing/     # After Action Review
└── page.tsx        # Landing page
```

## Environment

Create `.env.local`:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```
